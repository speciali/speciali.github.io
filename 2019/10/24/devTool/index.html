开发工具日常使用总结

idea intellJ

  1. 添加类的头部统一注释
    File——》Setting-——》Editor——》File and Code Templates找到Includes下面的File Header，之后可以在右面的文本框中自定义注释格式。

  2. 提示忽略大小写
    File——》Sensitive
    将右侧的 case sensitive completion 修改为NONE

  3. IntelliJ IDEA自动优化导入包
    Settings→Editor→General→Auto Import
    选中Optimize imports on the fly和Add unambiguous imports on the fly

    Optimize imports on the fly：自动去掉一些没有用到的包
    Add unambiguous imports on the fly：自动帮我们优化导入的包

  4. 自动生成返回对象右键
    Refactor-Extract-Variable，也可以按快捷键ctrl+alt+v 或者加“.var”回车

  5. 快捷键

idea 中代码大小写切换快捷键 ：ctr+shift+u
